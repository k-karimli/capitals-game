# Capital City Game - User Guide

**Purpose:** Test your geography knowledge by identifying capital cities from a list of five cities.

**Access:** Public - no login required

## Powered by Manus

Built with cutting-edge web technologies for a smooth, responsive experience. The frontend leverages **React 19** with **TypeScript** for type-safe component architecture, styled with **Tailwind CSS 4** for modern, utility-first design, and enhanced with **shadcn/ui** component library for polished, accessible UI elements. **Deployment:** Auto-scaling infrastructure with global CDN ensures fast load times worldwide.

## Using Your Website

The Capital City Game challenges you to identify capital cities while tracking your score. Start by clicking "Start Game" on the welcome screen. Each round presents five cities where exactly one is a capital city. Click the city you believe is the capital. If correct, your score increases and a new round begins automatically. The game continues until you select a non-capital city, at which point the game over screen displays your final score and reveals the correct answer. Click "Play Again" to restart and try to beat your previous score.
